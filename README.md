# AB WP Bits

A WordPress plugin providing useful bits and utilities.

## Description

AB WP Bits is a WordPress plugin designed to extend WordPress functionality with custom features and utilities.

## Installation

1. Upload the `ab-wp-bits` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
